    1. > ./{{vm.cliScriptName[vm.choosenOsType]}}

    subdirectory_arrow_right Displays the arguments and parameters supported by the UniFlash Command Line Interface
    3. > ./{{vm.cliScriptName[vm.choosenOsType]}} <customArguments>
    subdirectory_arrow_right Provide a list of arguments and parameters supported by the UniFlash Command Line Interface. For example, you can use it to perform Flash operations, enable logging, specify core information, etc.
